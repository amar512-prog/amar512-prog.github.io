PyTorch quickstart notes for getting moving fast

PyTorch is an open-source machine learning framework widely used for deep learning research and production work. This page keeps the basic "what is it?" and "how do I install it?" answers in one place.

What is PyTorch?

PyTorch is an open-source machine learning framework designed for deep learning applications and developed by Facebook's AI Research Lab (FAIR). It is especially popular because it is flexible, Python-friendly, and pleasant to use for both experimentation and production pipelines.

Installation

Two common installation paths are still the easiest to remember:

Install with Anaconda

If you manage environments with `conda`, this is a straightforward starting point. 

conda install pytorch torchvision -c pytorch

Install with pip

If you prefer a lighter Python setup, install with `pip` instead. 

# Python 3.x
pip3 install torch torchvision
These commands are enough for a quick start note, but in practice you should still confirm the latest CUDA or CPU-only install guidance from the official PyTorch website when setting up a real environment.
