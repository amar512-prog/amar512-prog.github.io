Contest Coding Tips

The original note was written on 19 Dec 2022 and kept only a few short reminders. They are simple, but they still reflect the right instinct for performance-sensitive coding.

  • Use addition in place of multiplication when that transformation is valid and cheaper.
  • To count the number of 1 bits in an input, use __builtin_popcount(unsigned int x).
  • Be careful about duplicate computation. Repeating the same expression in a hot loop is an easy way to waste time.
The spirit of the original reminder was straightforward: in contests, small inefficiencies add up quickly, especially inside repeated loops or heavily used helper logic.
