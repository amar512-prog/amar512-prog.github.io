Contest Coding Tips

These reminders were first written down on 19 Dec 2022. They are simple, but they still reflect the right instinct for performance-sensitive coding.

  • Use addition in place of multiplication when that transformation is valid and cheaper.
  • To count the number of 1 bits in an input, use __builtin_popcount(unsigned int x).
  • Be careful about duplicate computation. Repeating the same expression in a hot loop is an easy way to waste time.
The core idea is straightforward: in contests, small inefficiencies add up quickly, especially inside repeated loops or heavily used helper logic.

This note is short on purpose. It is less a full tutorial and more a small “remember these when the clock is running” card for time-sensitive coding work.
